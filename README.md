# expense_tracker_mern

A full stack expense tracker

## Quick Start

#### Install dependencies
```
 npm install
 cd client npm install
 cd ..
 ```
 
#### Run front and backend
```
npm run dev
```

#### Backend only
```
npm run server
```

#### Frontend only
```
npm run client
```

#### Build client
```
 cd client
 npm run build
```

#### Prod
```
npm start
```
